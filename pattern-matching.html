Pattern Matching

Rust has no switch statement.


Instead, match is used


The match keyword allows for powerful control flow.


A basic match can function almost identically 



        

        

        

            
Using match in place of switch


let x = 5;

match x {
    1 => println!("one"),
    2 => println!("two"),
    3 => println!("three"),
    4 => println!("four"),
    5 => println!("five"),
    _ => println!("something else"),
}
