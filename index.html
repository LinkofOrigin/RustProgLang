Intro to Rust

Hi there, and welcome to Rust!


  • Rust is a systems programming language (like C)
  • Three main goals: safety, speed, and concurrency
  • Has many compile-time safety checks that don't produce runtime overhead
  • Rust has three versions: stable, beta, and nightly
  • Rust uses Cargo as a package manager (think "make")

Now let's look at some Rust code


fn main() {
// Mutability
let x = 1;
// x = 2; // error

let mut y = 5;
y = 10; // There we go!

// Primitives
let a = true; // boolean
let b = 'b'; // char, four bytes - Unicode
let c = 123; // integer
let d = 1.23; // floating point
let e = "Hello, World!"; // string
let f = [1, 2, 3]; // array (fixed size)
let g = (1, "Hello"); // tuple (fixed size)
let h = vec![1, 2, 3]; // vector (dynamic size)

fn foo(x: i32) -> i32 { x } // function
let x = foo; // function pointer
}
